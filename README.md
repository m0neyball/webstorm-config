How to install
---

1. Clone it on your desktop
2. **File** -> **Import Settings**
3. import **webstorm_settings.jar** file
4. **WebStorm** -> **Preferences...** -> **Editor** -> **Code Style**
5. **Scheme** use **MoneyballCodeStyle**
6. Click **Apply**
6. Click **OK**
